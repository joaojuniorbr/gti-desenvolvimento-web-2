JQuery Plugins

Exercícios de aplicação de plugins JQuery

Exercício 1 - Galeria

Na página "Galeria", implemente uma galeria de imagens baseada em jQuery com um plugin a sua escolha.

Exercício 2 - Tabela

Na página "Tabela", utilize um plugin que permita ao usuário editr a ordem dos dados da tabela sem precisar recarregar a página ou reescrever a tabela, clicando na coluna de título pela qual deseja realizar a ordenação.

Exercício 3 - Formulário

Na página "Formulário", utilize um plugin de sua preferência para adicionar um medidor de força/complexidade de senha ao campo Senha.
